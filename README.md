# Metro Vancouver Crash Analysis – SQL ETL + Tableau Geospatial Dashboard
Insights from Reported Crashes in Metro Vancouver for 2018 - 2022: A Visual Analysis Using Tableau.
Analyzed 5 years of Metro Vancouver crash data using SQL (joins, CTEs, window functions) to clean, transform, and prepare datasets for geospatial visualization. Built an interactive Tableau dashboard to identify high‑risk intersections, collision hotspots, and temporal patterns, supporting data‑driven decisions for urban planning and road‑safety improvements.
